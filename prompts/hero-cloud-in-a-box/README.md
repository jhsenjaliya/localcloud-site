# Cloud In A Box Hero Prompt Pack

This prompt pack is for generating a homepage hero image for LocalCloud.

## Shared Art Direction

Use case: `stylized-concept`

Asset type: homepage hero image

Primary request:
A sleek isometric illustration of a miniature cloud data center sitting on a developer's desk next to a laptop. The tiny data center is a translucent glass cube glowing with soft blue and teal light, containing miniaturized cloud infrastructure such as database cylinders, message queues, storage buckets, and API gateways, all connected by flowing light trails.

Scene/backdrop:
Minimal modern developer desk, dark matte surface, ambient blue-purple backlighting, subtle coffee cup, one small potted plant, dark cinematic background.

Subject:
The glass cube is the "cloud in a box" container. A single cable connects the cube to the laptop. Above the cube, fifteen floating holographic service glyphs orbit gently like satellites.

Style/medium:
Clean 3D isometric illustration, premium product-render feel, Pixar-quality materials, soft volumetric lighting, shallow depth of field on the edges.

Composition/framing:
16:9 landscape, hero-ready framing, cube and laptop as primary focal points, enough breathing room around the scene, no text.

Lighting/mood:
Futuristic but approachable, crisp glow from blue to teal to emerald accents, dark environment with soft contrast and luminous detail.

Color palette:
`#0a0a0f` background, electric blue, teal, emerald, subtle lavender edge light.

Materials/textures:
Translucent glass cube, polished metal laptop, matte desk, holographic service glyphs, soft bloom, clean reflections.

Constraints:
- No text
- No people
- No brand wordmarks
- No cluttered desk
- No flat 2D illustration look
- Service icons should read as simple cloud-service pictograms, not literal branded logos

Avoid:
- Cartoonish proportions
- Harsh neon overload
- White studio background
- Generic stock-office look
- Busy UI text on the laptop

## Notes

- The prompts are optimized for a homepage hero render, not a poster.
- The orbiting service symbols should suggest breadth of services without introducing readable text.
- The laptop screen should show a clean dark console with green status dots only.
